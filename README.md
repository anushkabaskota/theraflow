# TheraFlow

TheraFlow is an AI-assisted therapy supervision and session management platform designed to support psychology trainees working under licensed supervisors. The system focuses on structured workflows, ethical AI usage, and human-in-the-loop oversight.

---

## Features

- Supervised onboarding and approval workflows for trainees and supervisors  
- AI-assisted intake structuring and session summarization (assistive, not autonomous)  
- Role-based access control and session lifecycle management  
- Audit-friendly records with full workflow traceability  

---

## Tech Stack

- **Cloud**: Google Cloud Platform (GCP)  
- **Database**: Firestore  
- **AI**: Large Language Models for assistive documentation  
- **Process**: Agile sprint-based development
